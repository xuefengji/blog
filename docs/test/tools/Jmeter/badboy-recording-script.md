**重点**: 此方式只能录制 Web 端

## 脚本录制

**第一步**：在输入栏中输入要录制的地址

**第二步**：打开录制按钮

**第三步**：进入页面开始录制

**第四步**：录制结束后，停止录制

![badboy](./images/badboy2.png)

录制好后左边的 Step1 下会有 HTTP 请求

![badboy](./images/badboy3.png)

**第五步**: 将录制好的脚本导出为 JMeter 脚本

![badboy](./images/badboy4.png)

**第六步**：在 JMeter 中打开这个脚本并进行回放

![badboy](./images/badboy5.png)

（完）
