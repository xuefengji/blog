# 面试心理学

+ 主导权

  掌控整体面试节奏的权力

  在回答一些不会的问题时，可以反问面试官

  例：问：请问有做过接口自动化测试

  答：一般做 web 自动化比较多，接口自动化没有接触过，请问贵公司现在是有在做接口自动化这块嘛？

  但会存在以下几种结果：

  + 回复
  + 不回复
  + 产生抗拒：不要尝试了

+ 交流

  一定要清楚

+ 口头禅

  尽量不要说

+ 肢体语言

  可以适量有

  身体语言：

  + 身体后仰

    表明面试官对当前的话题不感兴趣，这时候可以结束这个话题

  + 撇嘴/抿嘴、摇头

    表示面试官对你的观点不赞同

  + 十指交叉

    单独动作：表示面试官有自己的观点，但在听你在说观点

    加上抿嘴：表示不认同你的观点
